# bristle
A simple web gui for [snort](https://www.snort.org/). Snort is an Intrusion Detection System and alarms when dangerous activity is happening in your network.

## Features
 - List events and see their payload
 - View activity over different time periods in charts
 - Top 5 statitics

The project is in an early state and more features are planned. For example filter events and bunch duplicate events

## Getting started
Prerequisities: [snort](https://www.snort.org/), [barnyard2](https://github.com/firnsy/barnyard2), php, mysql, webserver
 1. Download the files to your www directory.
 2. Copy conf.php.example to conf.php
 3. Change the content in conf.php in accordance with your database setup. Bristle is using the same database as barnyard2.

Don't hesitate asking questions about install or other problems.
